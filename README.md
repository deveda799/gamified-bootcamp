# 游戏化训练营 MVP V1

面向知识付费训练营的移动端优先游戏化学习平台。

## V1 冻结范围

- 登录
- 课程学习
- 签到
- 作业提交
- 积分
- 等级
- 徽章
- 排行榜
- 后台查看

不包含 AI、海报、证书、案例库、支付、小程序、后台编辑器和老师点评工作台。

# 阶段 9：后台只读页接入

## 本阶段目标

在不新增 MVP V1 冻结范围之外功能的前提下，将后台查看页面接入统一 API 返回结构。

## 已完成内容

1. 新增后台只读 read model：
   - `lib/queries/admin-read-models.ts`

2. 新增后台只读客户端组件：
   - `components/admin/AdminApiState.tsx`
   - `components/admin/AdminOverviewClient.tsx`
   - `components/admin/AdminStudentsClient.tsx`
   - `components/admin/AdminSubmissionsClient.tsx`
   - `components/admin/AdminCheckInsClient.tsx`

3. 后台 API 接入 read model：
   - `app/api/admin/students/route.ts`
   - `app/api/admin/submissions/route.ts`
   - `app/api/admin/check-ins/route.ts`

4. 后台页面接入 API 客户端组件：
   - `app/admin/overview/page.tsx`
   - `app/admin/students/page.tsx`
   - `app/admin/submissions/page.tsx`
   - `app/admin/check-ins/page.tsx`

## V1 范围控制

本阶段仅保留后台查看能力，不包含编辑、删除、人工调整积分、作业点评工作台、AI 点评等 V1.1 Backlog 功能。

# 游戏化训练营产品总纲

## 文档定位

本文档是项目产品范围的统一入口。详细需求、原型和冻结规格保留在 `docs/产品/`。内容发生冲突时，按以下优先级处理：

1. 本文档确认的产品边界。
2. `docs/产品/游戏化训练营平台-MVP V1冻结版开发规格.md`。
3. `docs/产品/游戏化训练营平台-产品方案.md`。
4. `docs/产品/游戏化训练营平台-低保真原型规格.md`。

## 产品定位

面向知识付费训练营的移动端优先游戏化学习平台，通过课程、任务、签到、积分、等级、徽章和排行榜形成可追踪的学习闭环。

## MVP V1 范围

- 登录
- 课程学习
- 签到
- 作业提交
- 积分
- 等级
- 徽章
- 排行榜
- 后台只读查看

## V1 非目标

- AI 点评或 AI 教练
- 老师点评工作台
- 案例库
- 海报和证书
- 支付
- 微信小程序
- 补签
- 人工调整积分
- 后台业务数据编辑

## 技术原则

- 页面和业务逻辑不直接依赖具体数据供应商。
- 关键写入必须由服务端执行，并保证原子性和幂等性。
- 敏感密钥只保存在本地或部署环境变量中，不写入源码和文档。
- 只实现当前 Sprint 的必要范围，不提前扩展 V1.1 功能。

# 项目路线图

## 当前里程碑：MVP V1 真实数据闭环

1. 建立应用端口、统一错误和架构边界检查。
2. 完成可迁移 PostgreSQL Schema、事务函数和种子数据。
3. 完成 Supabase Auth、用户初始化和权限控制。
4. 接通学生首页、课程路径和课节真实读取。
5. 接通签到和课程完成原子写入。
6. 接通作业读取、草稿、附件上传和正式提交。
7. 接通积分、等级、徽章和排行榜真实数据。
8. 接通后台只读数据与组织权限。
9. 清理旧入口并执行测试、构建和真实核心链路验收。

## 部署里程碑

1. Vercel + Supabase Cloud 测试部署。
2. 国内云预生产环境和正式域名验证。
3. 视业务需要迁移到国内 PostgreSQL、对象存储和认证服务。

## 后续版本

AI 点评、老师点评、案例库、海报、证书、支付、小程序、补签和人工调分统一进入 V1.1 或更后版本，不进入当前里程碑。

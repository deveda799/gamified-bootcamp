# 项目目录结构

```text
gamified-bootcamp/
├── app/                         Next.js App Router 页面与 API
├── components/                  UI、学生端、后台组件
├── lib/                         Supabase、鉴权、领域逻辑、查询封装
├── supabase/                    数据库迁移与种子数据
├── tests/                       Node test 测试
├── docs/                        项目说明文档
├── public/                      静态资源
├── package.json                 依赖与脚本
├── tailwind.config.ts           A 方案视觉令牌
├── next.config.ts               Next.js 配置
└── tsconfig.json                TypeScript 配置
```

第一批开发只创建 V1 必需骨架：登录、课程、签到、作业、积分、等级、徽章、排行榜、后台查看。

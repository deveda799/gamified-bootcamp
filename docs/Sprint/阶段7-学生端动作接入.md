# 阶段 7：学生端动作接入

## 已实现

- 首页签到卡接入 `POST /api/check-ins`。
- 课节页“完成学习”接入 `POST /api/lessons/[lessonId]/complete`。
- 作业页接入：
  - `POST /api/submissions/draft`
  - `POST /api/submissions/submit`
- 新增统一动作反馈：
  - 成功：显示获得积分和新徽章。
  - 重复完成：显示已完成提示。
  - 失败：显示接口错误信息。

# 阶段 5：课程完成闭环

## 已实现

- `POST /api/lessons/[lessonId]/complete` 从占位改为服务端写入。
- 使用 Supabase Auth 获取当前用户。
- 使用 service role 服务端执行进度、积分、徽章写入。
- 同一学员同一课节重复完成不会重复发积分。
- 首次完成课节写入或更新 `lesson_progress`。
- 完成后写入 `point_ledger`，event key 为：

```text
lesson_completed:{enrollmentId}:{lessonId}
```

- 完成后根据 V1 徽章规则检查新徽章，例如完成第一节课触发“觉醒者徽章”。

## 相关文件

- `lib/services/lesson-completion.ts`
- `lib/repositories/lesson-completion.supabase.ts`
- `app/api/lessons/[lessonId]/complete/route.ts`
- `tests/lesson-completion-service.test.ts`

## 暂不包含

- 不包含视频播放防拖拽。
- 不包含课程编辑后台。
- 不包含老师点评。
- 不包含 AI、海报、证书、案例库。
- 不包含人工调整积分。
